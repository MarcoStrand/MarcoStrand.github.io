JavaScript学习笔记——数组

发表于2023-01-29|更新于2023-01-31
|阅读量:

数组

1.数组的概念

一组数据的集合，可以存储在单个变量下

2.创建数组

方法：

  1. 利用new创建数组

    var arr = new Array();

  2. 利用数组字面量创建数组

​ //1.使用数组字面量方式创建空的数组

​ var 数组名 = [];

​ //2.使用数组字面量方式创建带初始值的数组

​ var 数组名 =['小白','小黑','大黄','瑞奇'];

  • 数组里面的数据一定要用逗号分隔
  • 数组里面的数据称为数组元素
  • 字面量是使用最多的方式
  • 数组中的数据可以存放任意类型，数字型，字符型，布尔型，运算，函数等。

3.获取数组元素

数组的索引：

用来访问数组元素的序号（数组下标从0开始）

元素获取：

//定义数组

var arrStus = [1,2,3];

//获取数组中的第二个元素

alert(arrStus[1]);

4.遍历数组

如何把数组中全部元素取出来？

数组名[索引]的方法不够简便

遍历：就是把数组中的每个元素从头到尾访问一次

示例：

var arr = ['red', 'blue', 'green'];

​ for (var i = 0; i < 3; i++) {

​ console.log(arr[i]);

​ }

数组长度

数组名.length

注意：

  • 数组长度是元素的个数

计算数组的和及平均值

5.数组中新增元素

可以通过修改length长度及索引号增加数组元素

修改length长度

可以通过修改length属性达到扩容的目的

示例：

var arr = ['red', 'green', 'blue', 'pink'];

​ console.log(arr.length);

​ arr.length = 5; //把数组的长度修改为5，里面应该有五个元素

​ console.log(arr[5]); //undefined

修改数组索引新增数组元素

​ //修改索引号

​ var arr1 = ['red', 'green', 'blue', 'pink'];

​ arr1[4] = 'orange'; //追加数组元素

​ console.log(arr1);

​ arr1[1] = 'black'; //替换数组元素

​ console.log(arr1);

​ arr1 = 'white'; //不要给数组名赋值，否则里面的数组元素会被覆盖

​ console.log(arr1)
